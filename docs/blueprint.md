# **App Name**: Global Weather Watch

## Core Features:

- Country Search: Allow users to search for weather information by country name.
- Real-Time Weather Data: Display current weather conditions for the selected country, including temperature, humidity, wind speed, and a brief description of the conditions using generative AI as a tool.
- 7-Day Weather Statistics: Present weather statistics for the past 7 days, including high and low temperatures, precipitation, and wind speeds.
- Animated Weather Images: Display animated images that dynamically change based on the current weather conditions (e.g., sunny, cloudy, rainy, snowy).
- Interactive World Map: Implement an interactive world map to visualize weather conditions around the globe.
- Weather Data Source Integration: Fetch weather data from a reliable weather API and cache it for quick access.

## Style Guidelines:

- Primary color: Sky blue (#73C2FB) to represent clear skies and a fresh atmosphere.
- Background color: Light gray (#F0F0F0) for a clean and modern look.
- Accent color: Orange (#FFA500) to highlight important information and interactive elements.
- Body font: 'PT Sans', a humanist sans-serif for comfortable readability of weather information.
- Headline font: 'Space Grotesk', a geometric sans-serif, to create a techy, modern feel in headings.
- Use flat design, animated icons to visually represent different weather conditions, e.g., sun, clouds, rain.
- Incorporate smooth transitions and animations when the weather data is updated.
